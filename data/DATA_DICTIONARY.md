# Diccionario de Datos

## Dataset: mtsamples.csv

**Fuente:** [Kaggle - Medical Transcriptions](https://www.kaggle.com/datasets/tboyle10/medicaltranscriptions)

**Descripción:** Transcripciones médicas extraídas de mtsamples.com

**Tamaño:** 5,029 registros

---

## Columnas

| Columna | Tipo | Descripción |
|---------|------|-------------|
| `description` | texto | Breve descripción del caso médico |
| `medical_specialty` | texto | Especialidad médica (ej: Cardiology, Orthopedic) |
| `sample_name` | texto | Nombre/título del ejemplo médico |
| `transcription` | texto | Transcripción completa del documento médico |
| `keywords` | texto | Palabras clave asociadas al caso |

---

## Notas

- Los datos fueron extraídos de mtsamples.com
- Contiene información médica anonimizada
- Útil para tareas de clasificación de texto y NLP
